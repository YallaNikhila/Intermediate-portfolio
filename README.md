# Personal Portfolio Website

## Project Overview

This project is a Responsive Personal Portfolio Website created using HTML5 and CSS3 as part of the Developers Arena Internship Week 2 Task.

The main purpose of this project is to learn how to design and style a webpage using CSS concepts such as selectors, colors, Flexbox, spacing, hover effects, and responsive design.

The portfolio website contains different sections including:
- About Me
- Skills
- Contact Form
- Navigation Bar
- Footer

The project is designed to be simple, responsive, and beginner-friendly.

---

# Project Objectives

The objectives of this project are:

- Learn the fundamentals of CSS styling
- Understand how to connect external CSS files with HTML
- Create responsive webpage layouts
- Use Flexbox for arranging elements
- Apply hover effects and transitions
- Build a visually appealing portfolio website
- Practice responsive web design for mobile devices

---

# Technologies Used

This project was developed using the following technologies:

- HTML5
- CSS3
- Google Fonts
- Flexbox
- Media Queries
